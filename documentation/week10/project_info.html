Class diagram

The class Animation has two fields for storing the list of original keyframes and the interpolated keyframes. The functions implemented are used for adding new keyframes, select one keyframe from the list, remove the selected keyframe and reproduce the animation.

The class Keyframe has five fields implemented where the name, time, position, rotation and scale are stored. The functions implemented in this class are the constructor, a displayer and an editor.
